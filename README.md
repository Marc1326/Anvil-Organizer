# Anvil Organizer

Mod-Organizer-GUI nach MO2-Vorbild (Dark Theme). Platzhalter-GUI für spätere Game- und Mod-Logik.

- **Python:** 3.11+
- **GUI:** PySide6

## Einrichtung (optional: virtuelle Umgebung)

```bash
cd "/home/mob/Projekte/Anvil Organizer"
python -m venv .venv
.venv/bin/pip install -r requirements.txt
```

## Start

```bash
cd "/home/mob/Projekte/Anvil Organizer"
python main.py
# oder mit Venv:
.venv/bin/python main.py
```

Alle Buttons, Dropdowns und Tabs sind Platzhalter und geben nur `TODO`-Ausgaben aus.
